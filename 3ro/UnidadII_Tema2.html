HTML - HyperText Markup Language (Lenguaje de Marcado de Hipertexto)


Es el lenguaje estándar que se utiliza para crear y estructurar el contenido de las páginas web. A través de etiquetas, HTML permite definir elementos como títulos, párrafos, imágenes, enlaces, listas y tablas, entre otros. Cada elemento en HTML se escribe entre etiquetas, y muchas de ellas tienen atributos que ofrecen información adicional o modifican su comportamiento.

Para realizar un documento html es importante tener en cuenta algunas recomendaciones que ayudan a escribir código claro, ordenado y comprensible:

  • Indentación adecuada: se refiere al uso de espacios o tabulaciones al comienzo de una línea de código para organizar visualmente su estructura. Aunque en HTML no es obligatoria para que el código funcione, facilita su lectura y comprensión, permite identificar rápidamente jerarquías y relaciones entre elementos y mejora el mantenimiento del sitio, especialmente en proyectos con muchos archivos o desarrolladores.
  • Organización de archivos: permite mantener el proyecto ordenado, facilitar su comprensión y evitar errores comunes como rutas incorrectas o archivos perdidos. Una estructura clara suele incluir una carpeta principal con subcarpetas específicas para imágenes (img), hojas de estilo (css) y archivos JavaScript (js), además de un archivo principal como index.html. Usar nombres descriptivos y sin espacios, agrupar los recursos por tipo y respetar las rutas relativas son hábitos que favorecen la escalabilidad del sitio y demuestran responsabilidad en el trabajo como desarrolladores.

Actividad práctica:

  1. Crear un documento html que contenga un encabezado, un párrafo, una lista ordenada con tres ítem y una imagen. El primer ítem de la lista debe contener un enlace a otro documento html.
