Tema 1: Escalas y medición

Medir significa comparar una característica de un objeto o fenómeno con una unidad de referencia para saber cuánto tiene de esa característica.

Las mediciones nos permiten comparar, tomar decisiones y entender mejor el mundo.

¿Qué es una magnitud?

Una magnitud es una propiedad que se puede medir.

Magnitud Unidad
Longitud metro (m)
Masa kilogramo (kg)
Tiempo segundo (s)
Temperatura °C
Volumen litro (L)

Mediciones en sistemas tecnológicos

Muchos sistemas tecnológicos funcionan gracias a mediciones realizadas por sensores.

  • Entrada: sensor mide un dato
  • Procesamiento: el sistema analiza la información
  • Salida: el sistema genera una respuesta
Ejemplo: un sensor mide la temperatura y el sistema decide encender un ventilador.
