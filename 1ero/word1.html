Unidad III: Ofimática

¡Bienvenidos a una nueva unidad! A lo largo de estas clases vamos a conocer y usar herramientas ofimáticas, es decir, programas informáticos que nos ayudan a crear, editar y organizar distintos tipos de documentos: textos, tablas, presentaciones, entre otros. Estas herramientas son muy útiles tanto para el estudio como para la vida diaria.

¿Qué es la ofimática?

La ofimática es el conjunto de programas y técnicas que usamos en la computadora para realizar tareas de oficina o escolares, como escribir un trabajo práctico, calcular un presupuesto o armar una presentación.

Herramientas principales:

  • Procesador de texto (Microsoft Word): para escribir y dar formato a documentos.
  • Hoja de cálculo (Excel): para organizar datos y hacer cálculos automáticos.
  • Presentaciones (PowerPoint): para crear diapositivas con texto, imágenes y gráficos.
  • Otras herramientas: correo electrónico, almacenamiento en la nube, diseño gráfico online, etc.

Primeros pasos en Microsoft Word

¿Qué es Word?

Microsoft Word es un procesador de texto que permite escribir, editar y dar formato a un documento.

¿Para qué se usa?

  • Redactar textos escolares (informes, cuentos, cartas, etc.).
  • Aplicar estilos de letra, insertar imágenes o tablas.
  • Mejorar la presentación de trabajos escolares.

¿Qué haremos hoy?

  1. Conocer los elementos básicos de la interfaz de Word.
  2. Transcribir el cuento "El Rey y el Bien".
  3. Aplicar configuración básica al documento.
  4. Guardar y subir a Drive.

Actividad en clase:

  1. Abrir Microsoft Word y transcribir el cuento: El Rey y el Bien.
  2. Agregar un párrafo al final del documento haciendo referencia a una enseñanza o moraleja.
  3. Aplicar la siguiente Configuración Básica:
    • Fuente: French Script MT (todo el texto).
    • Cambiar mayúsculas/minúsculas: Tipo oración (título)
    • Tamaño de fuente: Texto: 14. Título: 20.
    • Color: Texto: Azul. Título: Rojo.
    • Estilo: Título: Negrita y subrayado a gusto.
    • Alineación: Título: Centrado. Texto: Justificar.
    • Sangría: especial (primera línea de cada párrafo)
    • Interlineado: Espacio 1,5 (todo el texto).
    • Espaciado: Posterior 12ptos (todo el texto).
    • Margen: Superior e izquierdo: 3,5cm. Inferior y derecho: 2,5cm
  4. Guardar el documento con el nombre El rey y el bien, subirlo a tu carpeta de Informática en Drive.
